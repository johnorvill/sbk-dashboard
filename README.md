# SBK Ras- och Kenneldashboard

En enkel lokal dashboard som visualiserar ras- och kennelstatistik från
`SBK-Ras-Kennel-Statistik-2019-2023 (1).xlsx`.

## Starta

1. Generera datamängden:
   `python tools/convert_excel.py "H:\SBK-Ras-Kennel-Statistik-2019-2023 (1).xlsx"`
2. Öppna `index.html` i webbläsaren.

Dashboarden använder inga externa beroenden och kan därför köras helt lokalt.
